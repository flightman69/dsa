# Array
Array is a linear data structure that stores collection of elements of the same data type in a contiguous memory. It has constant time access. Each element has unique index number.

## Operations on Array:
  - **Traversal**: Iterating through the elements of an array.
  - **Insertion**: Adding an element to the array at a specific index.
  - **Deletion**: Removing an element from the array using it's specific index.
  - **Searching**: Finding an element in the array using it's index or value.

## Types of Arrays:
  - **One-Dimentional array**: A simple array with a single dimention.
  - **Multi-Dimentional array**: A more complex array with more dimentions, example a matrix.

## Applications of Arrays:
  - **Sorting**: Arrays are used to sort data.
  - **Create DS**: It can be used to create other data structures like stacks, queues.
  - **Representation of data**: Array can be used to represent data in various form such as matrix, table.
